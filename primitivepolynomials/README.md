# Primitive Polynomial Research Result Bundle

Main result:
The primitive-polynomial/LFSR-derived family of de Bruijn cycles is bounded by

F(n) <= (phi(2^n - 1)/n) / 2^(2^(n-1)-n)

and therefore tends to zero super-exponentially. In the sheaf/homology
framework, primitive polynomials are linear global sections; almost all
de Bruijn cycles are nonlinear global sections.
