
# (Co)homology Interpretation of Start/End Locality, Sheafification, and de Bruijn Cycles

## 1. Purpose

The previous notes reframed start/end locality as sheaf restriction:

\[
\rho_R(u)=\rho_L(v)
\]

for valid de Bruijn transitions.

This final note adds the corresponding homological and cohomological interpretation.

The guiding dictionary is:

\[
\boxed{
\text{cycles in a de Bruijn graph}
=
\text{1-cycles in graph homology}
}
\]

\[
\boxed{
\text{local consistency rules}
=
\text{cocycles / sheaf-compatible cochains}
}
\]

\[
\boxed{
\text{near-}G\text{ boundary defects}
=
\text{nonzero boundaries / coboundaries}
}
\]

\[
\boxed{
\text{failed lift from pattern to symbols}
=
\text{cohomological obstruction}
}
\]

## 2. Graph homology of a directed de Bruijn graph

Let

\[
B(q,n)=(V,E)
\]

be the de Bruijn graph.

For homology, view it as an oriented 1-dimensional CW-complex:

- \(C_0\): free abelian group generated by vertices;
- \(C_1\): free abelian group generated by directed edges.

So

\[
C_0=\mathbb Z[V],
\qquad
C_1=\mathbb Z[E].
\]

The boundary map is

\[
\partial:C_1\to C_0
\]

defined on an edge \(e:u\to v\) by

\[
\partial e = v-u.
\]

A 1-chain is a formal integer combination of edges:

\[
c=\sum_{e\in E} a_e e.
\]

Its boundary is

\[
\partial c
=
\sum_{e:u\to v} a_e(v-u).
\]

Thus, at each vertex \(v\), the coefficient of \(v\) in \(\partial c\) is

\[
\text{incoming weight at }v-\text{outgoing weight at }v.
\]

So:

\[
\partial c=0
\]

means every vertex has balanced incoming and outgoing chain weight.

This is exactly the circulation condition.

---

## 3. de Bruijn cycles as homology classes

An Eulerian cycle in \(B(q,n-1)\) uses every edge once. Since every edge appears once and the path closes, its boundary is zero:

\[
\partial c=0.
\]

So an Eulerian de Bruijn construction is a 1-cycle.

Equivalently, a Hamiltonian cycle in \(B(q,n)\) is also a closed 1-chain at the vertex level.

Thus:

\[
\boxed{
\text{de Bruijn cycle}
=
\text{distinguished integral 1-cycle}
}
\]

more specifically, a 1-cycle whose support covers the required local sections exactly once.

---

## 4. Start/end locality as the boundary operator

For an edge

\[
e:x_1\cdots x_n\to x_2\cdots x_n a,
\]

the boundary

\[
\partial e
=
(x_2\cdots x_n a)-(x_1\cdots x_n)
\]

records the difference between end-locality and start-locality.

For a chain \(c\), the boundary \(\partial c\) records the net failure of incoming and outgoing local data to balance.

So:

\[
\boxed{
\partial c
=
\text{net start/end locality imbalance}.
}
\]

A balanced de Bruijn cycle has

\[
\partial c=0.
\]

A near-\(G\) object has

\[
\partial c\neq 0.
\]

---

## 5. Near-\(G\) defects as homological boundaries

Earlier we defined the imbalance

\[
\delta_H(v)=\deg_H^+(v)-\deg_H^-(v).
\]

For the edge chain

\[
c_H=\sum_{e\in E(H)} e,
\]

the boundary coefficient at \(v\) is

\[
(\partial c_H)(v)=\deg_H^-(v)-\deg_H^+(v).
\]

Therefore

\[
(\partial c_H)(v)=-\delta_H(v).
\]

So the near-\(G\) defect is exactly the negative of the graph-chain boundary.

Thus:

\[
\boxed{
\delta_H=0
\quad\Longleftrightarrow\quad
\partial c_H=0.
}
\]

and

\[
\boxed{
\delta_H\neq0
\quad\Longleftrightarrow\quad
c_H\text{ is not a cycle}.
}
\]

Positive and negative defect vertices are simply sources and sinks of the boundary chain.

## 6. Homology of the graph

For a connected graph with \(V\) vertices and \(E\) edges, the first Betti number is

\[
\beta_1=|E|-|V|+1.
\]

For a de Bruijn graph \(B(q,n)\),

\[
|V|=q^n,
\qquad
|E|=q^{n+1}.
\]

The underlying graph is connected, so

\[
\beta_1
=
q^{n+1}-q^n+1
=
q^n(q-1)+1.
\]

This counts the rank of the cycle space.

So \(B(q,n)\) has a large cycle space, and de Bruijn cycles are special integral cycles inside that space.

---

## 7. Cochains and local measurements

The cochain groups are

\[
C^0=\operatorname{Hom}(C_0,A),
\qquad
C^1=\operatorname{Hom}(C_1,A),
\]

where \(A\) is a coefficient group.

A 0-cochain assigns a value to each vertex.

A 1-cochain assigns a value to each edge.

The coboundary

\[
d:C^0\to C^1
\]

is defined by

\[
(df)(u\to v)=f(v)-f(u).
\]

So a coboundary measures the change of a vertex potential along edges.

In the de Bruijn context:

- a 0-cochain can encode a local word feature;
- a 1-cochain can encode transition cost, phase, parity, or defect;
- a cocycle is a transition measurement consistent around cycles.

---

## 8. Kernel basins as cohomological observables

The binary kernel theorem says that the eventual basin is determined by the central pair.

For binary words, define the central adjacent-XOR observable

\[
\omega(x)=x_m\oplus x_{m+1}.
\]

This is a 0-cochain on vertices:

\[
\omega:V_n\to \mathbb Z/2.
\]

It distinguishes:

- \(\omega=0\): constant basin type;
- \(\omega=1\): alternating basin type.

Under the kernel dynamics, all adjacent-XOR data contracts to this central observable.

So \(\omega\) is the surviving cohomological observable of the kernel flow.

For larger alphabets, the analogous observable is the equality relation:

\[
x_m=x_{m+1}
\]

or, with algebraic structure, the central difference

\[
x_{m+1}-x_m.
\]

Thus the basin decomposition is a cohomological measurement of the central overlap.

## 9. Sheaf cohomology viewpoint

Let \(\mathcal F\) be the word sheaf:

\[
\mathcal F([i,j])=A^{j-i+1}.
\]

Restrictions are substring maps.

A compatible family of local words is a 0-cocycle in the Cech sense: local sections agree on overlaps.

A failure of agreement on overlaps gives a 1-cocycle-like obstruction.

For example, if two local words \(u,v\in A^n\) are supposed to glue, the obstruction is

\[
\rho_R(u)-\rho_L(v).
\]

In a set-valued sheaf this is expressed as equality failure; in an algebraic sheaf it can be written as a difference.

Thus:

\[
\boxed{
\text{gluing consistency}
=
\text{cocycle condition}
}
\]

and

\[
\boxed{
\text{gluing failure}
=
\text{cohomological obstruction}.
}
\]

---

## 10. Pattern quotient and sheaf cohomology

The quotient by \(S_q\) gives a pattern sheaf:

\[
\mathcal P_q([i,j])=A^{j-i+1}/S_q.
\]

A pattern skeleton is a section of the quotient sheaf.

To lift a pattern skeleton to a concrete de Bruijn cycle, one must choose compatible symbol assignments in the fibers.

This is a lifting problem:

\[
\mathcal F \to \mathcal P_q.
\]

The obstruction to lifting lies in the failure to choose fiber assignments consistently around cycles.

In cohomological language:

\[
\boxed{
\text{pattern skeleton lifts}
\quad\Longleftrightarrow\quad
\text{associated obstruction cocycle is trivial}.
}
\]

So the sheafified construction is naturally a cohomology problem.

---

## 11. Fiber assignments as torsors

For a pattern \(p\) with \(r\) equality blocks, the fiber is

\[
\operatorname{Fib}(p)=
\{\text{injective assignments of }r\text{ blocks into }A\}.
\]

The group \(S_q\) acts on these fibers.

So each fiber behaves like an \(S_q\)-set, and often like a torsor after stabilizers are accounted for.

When moving along a pattern edge, overlap restrictions identify some blocks from one pattern with blocks from the next.

This induces transition constraints between fibers.

Around a closed pattern cycle, composing these transition constraints can produce monodromy.

If the monodromy is trivial, the local symbol assignments glue globally.

If not, the pattern cycle does not lift to a concrete de Bruijn cycle without refinement.

Thus:

\[
\boxed{
\text{monodromy around pattern cycles}
=
\text{lifting obstruction}.
}
\]

---

## 12. Cohomology of the quotient tower

The tower

\[
B(q,n)
\to
B(q,n+1)
\to
B(q,n+2)
\to\cdots
\]

has the edge-to-node relation

\[
E(B(q,n))\cong V(B(q,n+1)).
\]

After quotienting by \(S_q\), we obtain a tower of pattern graphs:

\[
\overline B(q,n)
\to
\overline B(q,n+1)
\to
\overline B(q,n+2).
\]

At each level:

- nodes are equality patterns;
- edges are pattern-compatible overlaps;
- fibers are concrete symbol assignments;
- cohomology measures obstruction to lifting pattern cycles.

So the tower is not merely combinatorial. It is a sequence of sheaf extension problems.

## 13. Relation to \(G_{n,m}\)

The original \(G_{n,m}\) idea measured local start and end degrees.

Homologically:

\[
\deg^+(v)-\deg^-(v)
\]

is the coefficient of a boundary chain.

So exact \(G_{k,k}\) corresponds to a cycle condition:

\[
\partial c=0.
\]

Near-\(G\) corresponds to a nonzero boundary:

\[
\partial c\neq0.
\]

Sheaf-theoretically:

- exact \(G\): local sections glue without boundary obstruction;
- near-\(G\): gluing defects appear;
- de Bruijn cycle: global section with full coverage.

Thus the three languages match:

\[
\boxed{
\text{balanced graph}
=
\text{homological cycle}
=
\text{sheaf-compatible gluing}.
}
\]

## 14. Binary special case: complement and parity

For binary alphabets, the complement operation

\[
C:0\leftrightarrow1
\]

gives a canonical symmetry.

The central parity observable

\[
x_m\oplus x_{m+1}
\]

is invariant under simultaneous complement:

\[
\overline{x_m}\oplus\overline{x_{m+1}}
=
x_m\oplus x_{m+1}.
\]

So the binary basin cohomology descends to the complement quotient.

This is why the binary quotient has only two endpoint types:

- constant type;
- alternating type.

In cohomological terms, the binary central parity is the surviving \(\mathbb Z/2\)-valued class.

---

## 15. Larger alphabets and \(S_q\)-symmetry

For a general alphabet with no arithmetic, the invariant is not a numeric difference but an equality relation:

\[
x_m=x_{m+1}
\quad\text{or}\quad
x_m\neq x_{m+1}.
\]

Under full \(S_q\)-symmetry, all equal pairs are equivalent, and all unequal pairs are equivalent.

So the quotient basin observable is again binary:

\[
\boxed{
\text{equal central pair}
\quad\text{versus}\quad
\text{unequal central pair}.
}
\]

If algebra is added, such as XOR or addition mod \(q\), this symmetry is reduced, and the cohomology coefficients become algebraic.

## 16. \(n=6\) and \(n=10\) as local cohomology witnesses

At \(n=6\), the short \(R_n\)-orbit

\[
001100\to011001\to110011\to100110
\]

contains all four central-pair types:

\[
00,\quad01,\quad10,\quad11.
\]

At \(n=10\), the short orbit

\[
0011001100
\to
0110011001
\to
1100110011
\to
1001100110
\]

does the same.

Since the basin observable is central-pair type, these short orbits contain all values of the local basin observable.

So they serve as finite witnesses of the local cohomological data.

In sheaf language:

\[
n=6,\ n=10
\]

give small local patches that realize every obstruction/basin type.

## 17. Practical construction interpretation

To construct de Bruijn cycles over a larger alphabet:

1. Build the quotient pattern graph.
2. Choose a pattern-level cycle or cycle cover.
3. Compute the fiber monodromy around the cycle.
4. Check whether the obstruction class vanishes.
5. If it vanishes, lift to concrete symbols.
6. If it does not, refine the pattern skeleton or add compensating cycles.

So:

\[
\boxed{
\text{cycle construction}
=
\text{homological cycle selection}
+
\text{cohomological obstruction cancellation}.
}
\]
