# De Bruijn Counting Addendum - Clean Bundle

This bundle adds three requested pieces:

1. Counting formula proof via BEST + homology.
2. Pattern-level counting decomposition as a quotient-sheaf/fiber-lift formula.
3. Constrained-system extension for DNA/RNA reconstruction, evidence-based prescription,
   and weather/atmospheric disturbance analysis.

It also includes:
- CSV of exact de Bruijn cycle counts for q=2,3,4 and n=1..7.
- LaTeX insert section.
- Small Python count script.

Important formula:
N(q,n)=(q!)^(q^(n-1))/q^n.
